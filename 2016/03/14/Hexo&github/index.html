Hexo + Github 打造博客站

在网上看到篇纪录hexo＋github建站的介绍，按照文章中的描述，自己实现了一遍。下面只简单说下在mac环境下的步骤。

Hexo配置

安装git

安装node.js

安装hexo

创建hexo 文件夹 和初始化

安装包依赖

生成&预览

到此时如果没有什么问题，那就可以打开http://localhost:4000进行预览

Github配置

注册账户（此处略）

创建repository

需要特别注意的是新建 repository 的名字.必须符合 github 的规则.

假如你的用户名为 yourname，则你的 repository 的名字必须为 yourname.github.io.

yourname 一定要记得替换为你 github 的用户名

http://yourname.github.io 就是访问路径 ，但是现在访问的话会有404页面出现，别着急，离成功很近了。

修改Hexo _config.yml

找到你在自己电脑初始化过的目录，里面有个 _config.yml 文件，编辑模式打开，进行如下修改

deploy:
type: git
repository: https://github.com/yourname/yourname.github.com.git
branch: master

所有的冒号有面都有个空格（路径除外),修改完成后保存文件。

生成&发布

如果上面的流程都没有问题的话，那就可以继续往下走

新文章的添加

只需要在 /your hexo root dir/sourcue/_posts/ 下面添加 markdown 文件即可

访问

在浏览器直接输入 http://yourname.github.io

常见问题

当执行 hexo deploy 时候，可能会有报错 ERROR Deployer not found: git?
